FAQ

Is API Tester free to use?

Yes, completely free. No signup, no limits. It runs entirely in your browser — no data is sent to any server besides the API endpoint you're testing.

Does API Tester support CORS-restricted APIs?

API Tester makes requests directly from your browser. APIs that restrict CORS may block requests. For testing such APIs, use a browser extension to disable CORS or test from a server-side environment.

Is my data stored or sent anywhere?

No. All request data stays in your browser's localStorage. Nothing is transmitted to any third-party server. Your history is private and can be cleared anytime.
